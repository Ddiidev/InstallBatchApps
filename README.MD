
# InstallBatchApps

Instala vários aplicativos no android de um vez.


## Apêndice

Bem eu criei isso porque no tempo estava testando umas coisas na maquineta da stone de dev, e não é tão simples passar os apps pra lá etc... Então eu fiz pra facilitar o processo e eu fiz em Euphoria porque nunca tinha usado a linguagem pra nada real que me ajudasse então resolvi fazer isso.(Faz um ano isso)

## Stack utilizada

**Lang:** Euphoria

**OS:** Windows
